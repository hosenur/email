# Agent Documentation

This document provides information about agents and how to use them in this project.

## Overview

Agents are autonomous systems designed to handle specific tasks or domains. They can be configured and customized to meet project needs.

## Getting Started

To get started with agents:

1. Review the agent configuration
2. Set up required dependencies
3. Initialize and run the agent

## Configuration

Agents can be configured through environment variables or configuration files. See the project setup documentation for more details.

### Agent Constraints

**Do NOT:**
- Run the dev server
- Build the project
- Attempt to install any packages
- Set background color of anything ever

These operations are restricted to maintain project stability and avoid unnecessary resource usage during agent operations.

## Usage

Refer to specific agent documentation for usage instructions and examples.
